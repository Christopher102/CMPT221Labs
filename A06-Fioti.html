My MySQL Reference Page

MySQL Database Commands

Command Description
USE Selects which database to use from
EXPLAIN Shows the column names of a table
CREATE OR REPLACE IF NOT EXISTS Creates either a table or database if it does not already exist
SHOW Shows the databases availible to you
DROP Deletes a table

MySQL Column Commands

Command Description
Attributes Attributes determine what values and type of data is allowed in a certain column
Modifers Modifiers create special variants of columns, such as primary keys
ALTER Allows you to alter columns in a table to either add or remove them, or to change values within them
ADD used with ALTER, adds columns or data
CHANGE Changes values in the column

MySQL Data Commands

Command Description
SELECT Grabs the listed columns from a table
FROM Chooses which table to use
WHERE Restricts what data based on certain values.
ORDER BY Sorts the table
INSERT Adds values into the table
DELETE Removes values based on a WHERE condition
UPDATE Overwrites previous values
