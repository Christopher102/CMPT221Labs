This lab will introduce us to basic HTML tags

The <DOCTYPE> tag is what sets the document as an HTML document
The <CHARSET> tag is used to set the encoding for the characters within HTML
The <TITLE> tag is used to set the "name" of the website that displays in the
browser bar
The <I> tag causes things to be italic!
The <B> tag causes things to be bold!
The <Strong> tag makes your text a strong emphasis, depending on user settings
The <HR> tag designates a line/break within the page

The <H1> through <H6> tags create "headers" for the page!

<Blockquote> creates indentation!

The <EM> tag creates emphasis!
